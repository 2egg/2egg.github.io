Java 运算符

算数运算符

运算符是用来计量数据的符号，数据可以是常量也可以是变量，被运算的操作的数据我们称之为操作符，算符运算符最常见的操作就是将操作数参与数学计算。

算数运算操作数必须是数值类型，

根据其运算数数量又分为一元运算符、二元运算符。

二元运算符 加减乘除 四则运算 取模则是求两个操作符相除后的余数。

运算符 含义 范例 结果
一 + 正号 +3 3
二 + 加号 2+3 5
String + 链接字符串 “hello”+”world“ helloworld
一 - 负号 int i = 3; -i -3
二 - 减号 3-1 2
二 * 乘 2*3 6
二 / 除 5/2 2
二 % 取模(取余数) 5/2 1
一 ++ 自增 int i=0;i++ 1
一 – 自减 int i=3；i– 2

++ 或 – 可以用于数值(操作数)

  • 之前：在赋值操作中，先对被++或–的操作变量进行+1 或者-1，然后再进行赋值。

  • 之后：在赋值操作中，先对被++ 或 – 的操作变量进行赋值，然后再进行自增或自减。

二元运算时，当两个操作数的数据类型不同，运算结果的数据类型和参与运算的操作数的数据类型中精度较高（或位数较长的）一致。

转换原则

低精度到高精度会自动转换，然而高精度到低精度则要进行强制转换

低精度——>高精度

byte、short、int、long、float、double、

数据长度

位：一个二进制数据 0 或 1 为1Bit （位）

字节：存储空间的计量单位、1Byte(字节)= 8 Bit,如数据库中的varchar 为字节

  1. Boolean类型的取值有：true、false 、 占1Byte
  2. Char 类型的数据：Unicode 字符、占2Byte
  3. Byte 一个字节（8位）（-128~127） -2的7次方到2的7次方-1
  4. short 两个字节（16位）（-32768 ~ 32767） -2的15次方到2的15次方-1
  5. int 四个字节（32位）（一个字长）（-21474483648 ~ 2147483647）-2的31次方到2的三十一次方减一
  6. long 8字节 64位 -2的63次方 到 2的63次方减一
  7. float 四个字节 32位 3.402823e+38 ~1.401298e-45 （e+38 是 数值乘以十的38次方 e-45 的 数值乘以十的负45次方）
  8. double 八个字节 64位 。。。。。。。
注意

数值计算中的语法现象为“晋升”、即：byte、short、和char、低于int 的数据类型 进行算数运算后，结果会自动提升为int类型。

两个char 类型运算时，会自动转换为 int 类型、或char 与其他类型进行运算时，也会自动传为int 类型的，再做其他类型的转换。

算数运算可以加入小口号“()”提高优先级，优先小小括号内的运算，再其他运算符运算。

算数运算前操作数变量必须赋值，反之，报语法错误。

关系运算符

关系运算符用于比较两个数值之间的大小，其运算结果为一个 逻辑类型(boolean布尔类型)的数值。

逻辑运算符

逻辑运算符要求操作数的数据类型为逻辑型，其运算结果也是逻辑型值。

位运算符

位运算是以二进制位为单位进行的运算，其操作数和运算结果都是整型值。
